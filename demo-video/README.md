# CertOps Demo Video

A programmatic product demo video built with [Remotion](https://www.remotion.dev/) for [CertOps](https://certops.com) — the Universal AI Verifier.

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm run dev
```

**Render video**

```console
npx remotion render
```

**Upgrade Remotion**

```console
npx remotion upgrade
```
